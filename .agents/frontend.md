## API Calls — Always Through BFF

The frontend never calls the backend directly. Every API call goes to the BFF.

```typescript
// apps/web/src/lib/api/bookings.ts
export async function approveBooking(
  bookingId: string,
  jwt: string,
  tenantSlug: string,
): Promise<void> {
  const res = await fetch(`${process.env.NEXT_PUBLIC_BFF_URL}/bookings/${bookingId}/approve`, {
    method: 'PATCH',
    headers: {
      'Authorization': `Bearer ${jwt}`,
      'X-Tenant-Slug': tenantSlug,
    },
  });
  if (!res.ok) throw new ApiError(res);
}
```

## Hotsite Pattern (public, unauthenticated)

The hotsite fetches the manifest from the BFF and renders modules server-side.

```typescript
// app/[slug]/page.tsx
export default async function HotsitePage({ params }: { params: { slug: string } }) {
  const manifest = await fetch(`${BFF_URL}/tenants/slug/${params.slug}`).then(r => r.json());

  return (
    <main style={{ '--primary': manifest.branding.primaryColor }}>
      {manifest.modules.map(module => (
        <HotsiteModule key={module.id} module={module} />
      ))}
    </main>
  );
}
```

## Dashboard Auth Guard

```typescript
// middleware.ts
export function middleware(request: NextRequest) {
  const token = request.cookies.get('jwt')?.value;
  if (!token && request.nextUrl.pathname.startsWith('/dashboard')) {
    return NextResponse.redirect(new URL('/login', request.url));
  }
}
```

## Locale and Money Display

- All UI copy in **pt-BR**
- Money: `R$ 1.234,56` format (use `Intl.NumberFormat('pt-BR', { style: 'currency', currency: 'BRL' })`)
- Dates: `dd/MM/yyyy` (pt-BR locale)
- Timezone: display in `America/Sao_Paulo` for all dates

```typescript
const formatted = new Intl.NumberFormat('pt-BR', {

  currency: 'BRL',

```
